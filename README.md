# BLog-Website

<h1>Blogging Website Demo</h1>

<h2>Description</h2>
A Blog Website demo with pages to display Posts, Categories, Single-post, Single-Page and earch Results.<br />


<h2>Languages and Utilities Used</h2>

- <b>HTML5</b> 
- <b>CSS</b>
- <b>BootStrap5</b>
- <b>JavaScript</b>

<h2>Environments Used </h2>

- <b>Windows 10</b>
